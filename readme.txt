Τα αρχεια για test βρίσκονται σε συμπιεσμένο φάκελο testing_data
Για να εκτελεστούν σωστά τα προγράμματα πρέπει να είναι 
το path στη μορφή testing_data/from_train και testing_data/from_test.
Τα αρχεια για train βρίσκονται σε συμπιεσμένο φάκελο all μεσα στο κάθε enron 
Για να εκτελεστούν σωστά τα προγράμματα πρέπει να είναι 
το path στη μορφή π.χ. enron/enron1/all. Τα σύνολα των συγκεκριμένων δεν περιέχουν τα 
test mails που έχουμε χρησιμοποιήσει.
